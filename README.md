# SAKIBU.DEV — Portfolio Website

Personal portfolio website for **Sakibu Mumuni**, a Backend Engineer based in Accra, Ghana, specializing in Python, Flask, and MongoDB.

## Tech Stack

- **HTML5** / **CSS3** / **Vanilla JavaScript** — no frameworks or build tools
- **Google Fonts** — Instrument Serif, DM Sans, JetBrains Mono
- Dark theme with neon lime (#c4f542) accent

## Sections

- **Hero** — Introduction and call-to-action (Download CV, Let's Connect)
- **About** — Bio, education background, and key stats
- **Tech Stack** — Python, Flask, MongoDB, PostgreSQL, Docker, AWS, Git, Linux, REST APIs
- **Education** — University of Ghana (BSc Computer Science & Mathematics) and ALX Africa
- **Projects** — Audita: a financial tracking solution for Ghana's education sector
- **Contact** — Email, phone, and social links

## Features

- Fully responsive (desktop, tablet, mobile)
- Scroll-reveal animations using Intersection Observer
- Mobile hamburger menu
- Toast notification system for external links
- Grain overlay effect and smooth CSS animations

## Getting Started

No installation required. Open `index.html` directly in a browser, or serve it locally:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```
